# Contributing to dotnet-sqlite-crud-generator

We welcome contributions to this project.

## How to Contribute

1. **Fork** the repository.
2. **Create** a feature branch: `git checkout -b feature/name`.
3. **Implement** your changes, keeping them focused.
4. **Run** tests: `dotnet test`.
5. **Commit** using [Conventional Commits](https://www.conventionalcommits.org/).
6. **Push** and submit a **Pull Request**.

## Guidelines

- Follow existing coding style (C# 10+, file-scoped namespaces, PascalCase for public members).
- Add XML documentation to all new or modified public APIs.
- Keep PRs focused.
- All CI checks (build, test, analysis) must pass.

## Reporting Issues

Use [GitHub Issues](https://github.com/sarmkadan/dotnet-sqlite-crud-generator/issues) for bugs or features. Please include environment details (.NET version, OS) and reproduction steps.
