# QueryBuilderBenchmarksExtensions

The `QueryBuilderBenchmarksExtensions` class provides a set of benchmarking utilities for evaluating the performance and behavior of query builder generation in the `dotnet-sqlite-crud-generator` project. These methods are designed to simulate real-world usage patterns, including repeated generation, query building, and cleanup cycles, to measure efficiency, resource consumption, and potential bottlenecks in the query builder infrastructure.
