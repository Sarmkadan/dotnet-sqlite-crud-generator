# Security Policy

We take the security of dotnet-sqlite-crud-generator seriously. 

## Supported Versions

Currently, the following versions are supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please follow these steps for responsible disclosure:

1. **Use GitHub's Private Advisory Feature:** We prefer that you report security vulnerabilities using GitHub's built-in private advisory feature.
2. **Contact via Email:** Alternatively, you can contact us directly by emailing [rutova2@gmail.com](mailto:rutova2@gmail.com).

Please provide a detailed description of the vulnerability and steps to reproduce it. We will acknowledge receipt of your vulnerability report within 48 hours and work with you to resolve it in a timely manner.
